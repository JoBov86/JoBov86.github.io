About Me

I started to learn coding seriously at the start of the Covid Lockdown when I was furloughed from my job at the time. I really enjoy it, I've mostly learnt by taking online courses and have made some mini projects to help me learn. I have mostly focussed on HTML, CSS and JavaScript. Although I have also taken a course on the React framework and will continue with this too.

I enjoy developing my skills and am really passionate about progressing with this career. When I'm not studying Web Development and code, I like to code games with my partner in our spare time. Recently, we have coded minesweeper and othello using Python and pygame. We are now building a huge quiz website with learning pages, quizzes and games. To build the site I have used React and have gained a lot of confidence and become much more skilled and knowledgeable about React.
